# CS628-PE-Anushree

The app uses resume details provided by the developer, such as name, contact info, education, technical skills, work history, and projects.All these details are written in the Resume.js component.

React takes the Resume component and turns it into HTML elements that show up in the browser.
The styles from Resume.css are used to make the page look nice, including the background color, fonts, spacing, and headings for each section.The App.js component brings in Resume.js and shows it as the main page of the app.

The result is a neat and professional resume webpage that displays all personal information in a clear way.
Users can see the resume by opening a web browser and running the React development server.The output has separate sections for education, skills, experience, and projects, all styled properly to make the resume easy to read and visually pleasing.
